# Image-Renaming-Tools-
A collection of Python scripts that rename media files using their original creation date, replacing Apple's cryptic photo and video filenames with something humans can actually understand.

pics.py is a helpful script that traverses the files in a directory and renames them in a universal/easily sortable format. 
imgRenamer.py does the same, but has a GUI.

These scripts happily improve a PC users IOS experience.
